**KRUD Shop** is a high-performance, feature-rich GUI shop system designed for modern Minecraft servers (1.21.x). Inspired by the stylish UI of **DonutSMP**, it brings a premium, immersive experience to your players with zero cost and no license keys.

---

## ✨ Features

*   **🎨 Stylish UI:** Beautiful gradients and clean layouts inspired by DonutSMP.
*   **🛒 Full GUI System:** Main menu, category menus, confirmation screens, and amount selectors.
*   **📦 Unlimited Categories:** Add as many categories as you want via simple YAML files.
*   **💰 Vault Integration:** Works seamlessly with any Vault-compatible economy plugin.
*   **⚡ High Performance:** Zero NMS code, ensuring compatibility across Bukkit, Spigot, Paper, Purpur, and Folia.
*   **🔄 Hot Reload:** Update your shops instantly with `/shop reload` without restarting the server.
*   **🆓 No License Keys:** Completely free to use, works offline, and requires no internet check.

## 🛠 Commands & Permissions

| Command | Permission | Description |
| :--- | :--- | :--- |
| `/shop` | `krudshop.use` | Open the main shop menu |
| `/shop <category>` | `krudshop.use` | Open a specific shop category |
| `/shop reload` | `krudshop.admin` | Reload all configurations |

---

## 🚀 Installation & Download

1.  **Download the latest release here:** [**GitHub Releases**](https://github.com/rinquicklyofficial/KRUD-Shop/releases/new)
2.  Ensure you have **Vault** and an economy provider (like EssentialsX or LiteEconomy) installed.
3.  Place the `KRUD-Shop.jar` in your `/plugins/` folder.
4.  Restart your server.
5.  Configure your items in `plugins/KRUD-Shop/categories/`.
